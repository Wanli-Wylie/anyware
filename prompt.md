## 一、总体要求

- **用途**：技术项目展示海报，面向计算科学、AI 与软件工程领域的研究者和工程师
- **风格**：深色主题（深蓝/近黑背景），科技感强，信息密度高，类似顶级学术会议 demo poster 或技术产品发布页
- **色彩体系**：背景 #0d1117 ~ #161b22；主色 teal/cyan (#58a6ff, #39d353)；辅助色 orange (#f0883e) 用于强调；文字 #e6edf3（亮灰白）
- **排版**：中文为主，代码片段保留英文。使用等宽字体呈现代码。标题粗大醒目，正文紧凑
- **尺寸**：纵向长海报，宽约 1200px，高约 2400–3000px，适合上下滚动浏览
- **整体气质**：严肃、精确、有技术深度，不花哨但有设计感。像一位知道自己在做什么的工程师的作品

### 区块 3：提取前后对比 — 地表反照率（Simsphere）

双栏对比，左侧 Fortran 原始代码，右侧 JAX 翻译版本，中间一个箭头 `→`。

左栏标题：**Fortran — 纠缠状态（9 个隐藏依赖）**

```fortran
subroutine ALBEDO(SOLSIN)
  use simsphere_mod, only: albg, albf,
      wgg, wmax, frveg, xlai, albdoe, sigf
  implicit none
  real :: SOLSIN

  ALBG = 0.25 - 0.20 * WGG / WMAX
  ALBF = 0.032/(0.1+0.1*SOLSIN)
    + (1-SOLSIN**2)*0.1
  SIGF = 1 - EXP(-0.4 * XLAI)
  ALBDOE = (SIGF*ALBF+(1-SIGF)*ALBG)
    +FRVEG + (1-FRVEG)*ALBG
end subroutine
```

右栏标题：**JAX — 纯函数，可微分，可向量化**

```python
def albedo(solsin,wgg,wmax,xlai,frveg):
    albg = 0.25 - 0.20 * wgg / wmax
    albf = 0.032/(0.1+0.1*solsin)
        + (1.0-solsin**2)*0.1
    sigf = 1.0 - jnp.exp(-0.4 * xlai)
    albdoe = (sigf*albf+(1-sigf)*albg)
        +frveg + (1-frveg)*albg
    return albdoe, albg, albf, sigf

# 向量化 + 自动求导：
batch = jax.vmap(albedo)
grad  = jax.grad(albedo, argnums=1)
```

视觉：两个代码块用深色卡片包裹，等宽字体，语法高亮。关键区别用颜色突出——左侧 `use simsphere_mod` 标红（纠缠），右侧 `jax.grad` 标绿（可微）。

### 区块 4：三个科学问题卡片（横排三列）

每张卡片：一个物理问题 + 来源代码路径 + 一句话解释 + 底部高亮梯度表达式。

**卡片 1：**
- 标题：**地面反射了多少阳光？**
- 来源：`simsphere · albedo.f90 · 40 → 16 行`
- 说明：地面反照率随土壤湿度变化（湿土更暗）。植被反照率随太阳角度变化。Beer 定律估算冠层消光。加权合成。
- 梯度（orange 底色标签）：`d(反照率)/d(土壤湿度)`

**卡片 2：**
- 标题：**水离开地表有多快？**
- 来源：`simsphere · vegflx.f90 · 25 → 18 行`
- 说明：Clausius-Clapeyron 饱和蒸气压。空气动力学阻抗网络中的潜热通量。地表温度·湿度精确。
- 梯度：`d(蒸发)/d(温度)`

**卡片 3：**
- 标题：**为什么旧雪和新雪看起来不一样？**
- 来源：`noah-MP · snowAlbedoClassMod.f90 · 60 → 12 行`
- 说明：反照率随时间指数衰减。新降雪使表面刷新。通过 lax.scan 运行 7 天轨迹，端到端可微分。
- 梯度：`d(第7天反照率)/d(老化速率)`
